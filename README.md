# 🎬 AI 사진 1장으로 영상 만들기 — Video Tools Guide

이 저장소는 **사진 1장으로 AI 영상을 생성할 수 있는 다양한 AI 영상 툴을 소개하는 웹페이지**입니다.  
각 툴은 링크와 함께 간단한 설명이 포함되어 있으며,

## 📚 소개

AI 기술을 활용하여 **사진 한 장만으로도 영상 생성이 가능한 시대**입니다.  
이 페이지는 교육, 워크숍, 강의 등에서 빠르게 소개할 수 있도록  
다양한 AI 기반 영상 생성 도구들을 한 곳에 모아 정리했습니다.
